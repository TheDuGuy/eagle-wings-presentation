# Eagle Wings Consulting - Brand Presentation

A modern, interactive brand identity presentation built with React, Tailwind CSS, and Framer Motion.

## Running the Project

The development server is running at: **http://localhost:5174**

Use arrow keys to navigate between slides.

## Features

- Smooth slide transitions with Framer Motion
- Interactive logo gallery (click to enlarge)
- Keyboard navigation support
- Modern glass morphism UI effects
- Responsive design

## Technologies

- React + Vite
- Tailwind CSS
- Framer Motion animations

## Slides

1. Title - Brand introduction
2. About - Company overview
3. Colors - Brand palette
4. Logos - 20 design concepts
5. Next Steps - Implementation plan
